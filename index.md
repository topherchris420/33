# Project 33: Low-Cost Folding-Fin Rocket System

**Vers3Dynamics | Applied Aerospace Research**

Project 33 is a college aerospace prototype showing how a folding-fin rocket system can be modeled, built, instrumented, and documented with low-cost desktop tools: ESP32 firmware, COTS sensors, FDM-printed parts, OpenRocket simulation, Fusion CAD, and a Python telemetry dashboard.

![OpenRocket 3D model](Simulation/OpenRocket_3D_View.png)

## What This Repository Contains

| Area | Contents |
|------|----------|
| CAD | Fusion 360 design archives and a NACA 4-digit airfoil generator |
| Simulation | OpenRocket model, side view, 3D view, and stability graph |
| Firmware | Rocket flight computer, launcher ground station, calibration sketches |
| Dashboard | Tkinter/matplotlib ground-control UI with automatic CSV telemetry logs |
| Docs | Wiring, architecture, safety, BOM, and testing evidence plan |

## System Overview

The Python dashboard talks to the launcher over UDP. The launcher owns the WiFi access point, physical arming controls, GPS/barometer/compass telemetry, and the UART bridge to the rocket. The rocket owns IMU roll sensing, canard servo output, and rocket-side arming/ignition state.

See [Architecture](docs/ARCHITECTURE.md) and [Wiring](docs/WIRING.md) for the full system map.

## Known Limits

This is an educational prototype. The repo does not yet include flight-test data, and the current stabilization loop is roll-axis focused. Treat live propulsion or ignition work as out of scope unless it is separately reviewed under qualified supervision and local rules.
