# Project 33: Low-Cost Folding-Fin Rocket System

## Vers3Dynamics | Applied Aerospace Research

Project 33 is an applied research initiative demonstrating that rigorous aerospace engineering principles—kinematic modeling, structural optimization, and deterministic control—can be executed on a desktop manufacturing platform.

Initially developed as a sub $100 proof-of-concept using COTS electronics and FDM 3D printing, this repository serves as the live engineering log for the system's optimization. It documents the transition from a basic bench prototype to a structurally sound, aerodynamically stable platform utilizing aerospace-grade materials and real-time embedded architectures.
